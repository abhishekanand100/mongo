#  CRUD

### Installing

Git clone

To run locally `sbt -jvm-debug 5005`

Environment Variable `$CONNNECTION`

with request body
`{"name":"preference", "clientId":123, "templateId":"Carousel Ad", "repeat":"daily"}`
